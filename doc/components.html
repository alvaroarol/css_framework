Components

Navigation bar

The navigation bar is declared with the class .nav. The menu elements should be placed inside links within an unordered list.

  • Item 1
  • Item 2
  • Item 3
<div class="nav">
    <ul class="container">
        <li><a href="">Item 1</a></li>
        <li><a href="" class="active">Item 2</a></li>
        <li><a href="">Item 3</a></li>
    </ul>
</div>

To keep the menu list within the maximum width of your page, you can give it a .container or .container-large class.
If you want to show which page is active in the menu, give the correspondent item the .active class.
Finally, you can easily change the color of the navigation bar with the .bg-x classes. This is explained further down.
It is also recommended to give the navigation bar a slight shadow with the class .shadow as explained in the effects page.

Buttons

Buttons are declared with the class .button. It is recommendend to use the <a> tag for buttons.

Button Longer button with more text 
<a href="" class="button">Button</a>
<a href="" class="button">Longer button with more text</a>

The color of the buttons can be changed with the .bg-x classes, like any other element. Scroll further down to see how.
It is also recommended to give buttons a slight shadow with the class .shadow as explained in the effects page.

Panels

A panel is declared with .panel.
Panels are made of 3 parts which can be used individually if needed : .panel-header, .panel-body and .panel-footer.

Panel header, usually the title

The body. The content usually comes here.

The footer can have multiple uses, like nesting action buttons.

Button Longer button with more text 
<div class="panel col col-12">
    <div class="panel-header">
        <p>The header</p>
    </div>
    <div class="panel-body">
        <p>The body</p>
    </div>
    <div class="panel-footer">
        <p>The footer</p>
    </div>
</div>

Notice how the panel element also has the classes col and col-12 to take up all the width available. You can change the width of the panel by setting a different col-x class.

Component colours

There are a few preset colours you can use to spice up your components, or any other element for that matter, thanks to the classes .bg-x where x is one of the available colours :
.bg-red .bg-blue .bg-lightblue .bg-yellow .bg-green

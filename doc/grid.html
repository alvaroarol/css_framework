Grid

Like many others, this CSS framework uses the standard 12 column grid structure. It is an easy way to arrange your page layout and, in case you've already used another popular framework like Bootstrap or Materialize, you will feel right at home.

Container

The content is kept inside a container which keeps it at a maximum width of 1200px (.container class).
An alternative wider container keeps it within 1600px (.container-large class).
No container 
<div>
    No container
</div>
Large container 
<div class="container-large">
    Large container
</div>
Regular container 
<div class="container">
    Regular container
</div>
Notice how all the containers have the same width if you lower the window width, since the container only defines a maximum width. Under that maximum width, the container will take up the entire window width.

Centered columns

In the case of not entirely filling a row, the columns can be centered with the class .row-center.
1
2
<div class="row row-center">
    <div class="col col-3">1</div>
    <div class="col col-3">2</div>
</div>

Right aligned columns

In the case of not entirely filling a row, the columns can be aligned to the right with the class .row-right.

2

</div>

Row wrap

In the case of filling up the row with more than 12 columns, the additional ones will still be placed on the same row and they all will be resized to fit the row.
col-6
col-6
col-6
If you want to avoid this behaviour, you can add the class .row-wrap to the row. This way, the extra columns will automatically be placed below, as if you created a new row.
col-6
col-6
col-6
<div class="row row-wrap">
    <div class="col col-6">col-6</div>
    <div class="col col-6">col-6</div>
    <div class="col col-6">col-6</div>
</div>

Vertical alignment

If your columns don't have the same height, the default is to vertically align them to the top. You can change this to the bottom or the center with the classes .row-vertical-bottom and .row-vertical-center.

Shadow

You can add a shadow to your elements with the class .shadow for a layered effect on your page. This is widely used on navigation bars and buttons for example.

Buttons with a a shadow
Panel
with a shadow 
<div class="row">
    <a href="" class="button col col-2 bg-lightblue shadow">Buttons</a>
    <a href="" class="button col col-1 bg-red shadow">with</a>
    <a href="" class="button col col-1 bg-yellow shadow">a</a>
    <a href="" class="button col col-2 bg-green shadow">a shadow</a>
    <div class="panel col col-6 border-blue shadow">
        <div class="panel-header bg-blue">
            Panel
        </div>
        <div class="panel-body">
            with a shadow
        </div>
    </div>
</div>

Shadow on hover

In some cases, you only want to show the shadow when an element is hovered by the cursor. This can be specified with the class .shadow-hover. This is a good way to give more emphasis on the element that is hovered.

Hover me
to see the shadow 
<div class="row row-center">
    <div class="panel col col-6 border-blue shadow-hover">
        <div class="panel-header bg-blue">
            Hover me
        </div>
        <div class="panel-body">
            to see the shadow
        </div>
    </div>
</div>
